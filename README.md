# Ecommerce-Price-Rating-Analysis

This project focuses on comparing product prices and customer ratings across three major Indian e-commerce platforms: Amazon, Flipkart, and Myntra. The objective is to identify differences in pricing strategies, analyze customer satisfaction trends, and understand how these factors influence buying decisions.

🎓 Academic Context

This project was developed as part of my MBA (Business Analytics) academic coursework. It demonstrates the practical application of data cleaning, transformation, and visualization using real-world datasets.

🛠 Tools & Technologies

* Python (Google Colab) – Data cleaning & transformation
* Microsoft Excel – Data merging & structuring
* Power BI – Data visualization & dashboard creation

 📂 Dataset
Source: GitHub & Kaggle
Combined dataset size: 10,00,000+ records
Data from Amazon, Flipkart, and Myntra were cleaned, standardized, and merged into a single dataset
Includes product names, prices, ratings, and categories 

🔄 Methodology
Data Collection
Data Cleaning
Data Transformation
Data Visualization
Analysis & Interpretation

📊 Key Insights
Amazon shows higher and more consistent customer ratings
Flipkart offers more competitive pricing, especially in electronics
Myntra performs strongly in fashion categories
No strong correlation between price and rating was observed
